## Data Processing

All data processing was accomplished in code.
The end result of the processing was a table with unique rows from each subject/activity combination.
The columns are the mean values of selected data columns from the original source.
The data columns selected were those referencing the function mean() or std().
See below for list of data columns selected

## Data Storage

The output is a single tab-delimited file with column headers named "tidydata.txt"
